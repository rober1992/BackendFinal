# coderhouse-ecommerce
## Final Project
App created with NodeJS, using Typescript. The server is working with Express.

## Prerequisites

- node >=14.0.0
- npm >=6.0.0

## Install

```sh
npm install
```

## Build

```sh
npm run build
```

## Usage

```sh
npm run start
```

## Run as dev

```sh
npm run dev
```
